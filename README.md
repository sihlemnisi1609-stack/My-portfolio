# Mfundo Mnisi Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript. It includes a looping animated background that alternates between a retro coder scene and moving binary rows.

## Live Demo

https://sihlemnisi1609-stack.github.io/My-portfolio/

## Features

- Full-screen animated canvas background
- Retro 80s coding scene with a box computer
- Automatic transition into moving binary rows
- Looping animation
- Clean content layout with project links
- freeCodeCamp profile link
- Responsive structure for desktop and mobile

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Canvas API

## Project Structure

- `index.html` — page structure and content
- `style.css` — layout, colors, and positioning
- `script.js` — animation logic
- `i_can_code_dark.svg` — illustration used on the page

## Running Locally

1. Download or clone the repository.
2. Open `index.html` in a browser, or use Live Server in Visual Studio Code.
3. The animation will run automatically.

## Notes

- The background animation is designed to loop continuously.
- The page content sits above the animated canvas so the text remains readable.
- The portfolio is intended to showcase my work, learning, and progress as an aspiring software developer.
